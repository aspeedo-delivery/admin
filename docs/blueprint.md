# **App Name**: Aspeedoo Admin Pro

## Core Features:

- Dashboard: Dashboard overview with key metrics (users, restaurants, revenue, orders).
- User Management: CRUD operations for user profiles including viewing wallet balance and orders.
- Restaurant Management: Restaurant management including CRUD operations, menu item management, and active status toggling.
- Food Orders: View, filter, and manage food orders with order item details.
- Courier Orders: Manage courier orders, change order status, and manually assign deliveries.
- Wallet Management: View wallet balances and transaction history, with the ability to credit/debit user wallets.
- Delivery Partner Management: List delivery partners, show delivered orders, and assign/unassign deliveries.

## Style Guidelines:

- Primary color: Vibrant blue (#4285F4) to reflect trust and stability.
- Background color: Light gray (#F5F5F5) for a clean and modern feel.
- Accent color: Soft green (#34A853) for success states and positive actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif font.
- Use a set of consistent and professional icons, preferably from Material UI or Font Awesome.
- A clean and organized layout with a sidebar navigation for easy access to different sections.
- Subtle animations for transitions and interactions to improve user experience.
